# Command Blocks

Purpose:
- Reusable confirmed command groups for child prompts
- Derived from live downstream `help` / `info`, not memory

Usage:
- Refer to block ids or file paths in child prompts
- Attach only the blocks needed for the current step
- Do not inline long command manuals when a block reference is enough

- English only
